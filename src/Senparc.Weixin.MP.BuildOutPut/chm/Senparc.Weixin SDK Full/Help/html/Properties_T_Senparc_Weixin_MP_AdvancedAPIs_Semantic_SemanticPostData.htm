The SemanticPostData type exposes the following members.

Properties
  NameDescription
appid
App id，开发者的唯一标识，用于区分开放者，如果为空，则没法使用上下文理解功能。 非必填
category
需要使用的服务类别，多个用,隔开，不能为空 必填
city
城市名称，与经纬度二选一传入 见说明，选填
latitude
纬度坐标，与经度同时传入；与城市二选一传入 见说明，选填
longitude
经度坐标，与纬度同时传入；与城市二选一传入 见说明，选填
query
输入文本串 必填
region
区域名称，在城市存在的情况下可省；与经纬度二选一传入` 见说明，选填
uid
用户唯一id（并非开发者id），用于区分该开发者下不同用户，如果为空，则没法使用上下文理解功能。appid和uid同时存在的情况下，才可以使用上下文理解功能。 非必填
