🧪 测试步骤

  1. 打开浮窗：点击页面上的绿色"Senparc.Weixin.AI"按钮打开AI助手浮窗
  2. 观察URL：注意当前页面的URL和iframe中加载的内容
  3. 关闭浮窗：点击浮窗右上角的关闭按钮
  4. 模拟URL变化：点击下方的"模拟URL变化"按钮改变页面URL
  5. 重新打开：再次点击绿色按钮打开浮窗
  6. 验证结果：观察iframe是否自动加载了新URL对应的内容
