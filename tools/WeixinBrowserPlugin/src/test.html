微信公众号开发文档

注意：这是一个测试页面，用于验证 Senparc.Weixin.AI Chrome 插件功能。
在真实的微信文档页面 (weixin.qq.com) 中，插件会自动检测并显示AI助手按钮。

接口调用说明

微信公众号为开发者提供了丰富的接口，用于实现各种功能。以下是一些常用的接口：

基础接口

  • 获取access_token
    获取接口调用凭据，用于调用其他接口
  • 获取微信服务器IP地址
    获取微信服务器IP地址列表

消息管理

  • 接收普通消息
    当用户发送消息给公众号时，微信服务器会将消息数据推送给开发者
  • 发送被动回复消息
    当用户发送消息给公众号时，开发者可以回复消息
  • 发送客服消息
    当用户和公众号有互动后，可以通过客服消息接口发送消息

用户管理

  • 获取用户基本信息
    通过OpenID获取用户基本信息
  • 获取用户列表
    获取关注者列表

示例代码

// 使用 Senparc.Weixin SDK 获取 Access Token
var accessToken = await AccessTokenContainer.GetAccessTokenAsync(appId);

// 发送文本消息
var result = await CustomApi.SendTextAsync(accessToken, openId, "Hello World!");

测试说明

要测试插件功能，请按以下步骤操作：

  1. 确保已安装 Senparc.Weixin.AI Chrome 插件
  2. 访问真实的微信文档页面：https://developers.weixin.qq.com/doc/
  3. 查看页面左上角是否出现绿色的"Senparc.Weixin.AI"按钮
  4. 点击按钮测试AI助手功能
提示：本插件专为微信开发者设计，集成了 Senparc.Weixin SDK 的 AI 助手功能， 可以帮助您快速查询微信接口信息和生成代码示例。
